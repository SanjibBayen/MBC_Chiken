# **App Name**: MBC Chicken Express

## Core Features:

- Product Catalog: Display product catalog with categories, variants (500g, 1kg, 1.5kg+), and market-dependent prices.
- Shopping Cart: Enable users to add products to a cart, manage quantities, and view order summary.
- Checkout Process: Allow users to enter delivery address, select payment method (COD, UPI, Cards), and place orders.
- Order Tracking: Provide a simple interface for users to view past orders and track their status.
- Scheduled Delivery: Offer scheduled delivery options based on the specified delivery timings (7 AM – 12 AM & 6 PM to 9 PM).
- Customer Reviews: Collect customer reviews and ratings for products.
- Smart Recommendations: Recommend related products based on items in cart and order history. The tool will assess when these recommendations are relevant, or just distracting.

## Style Guidelines:

- Primary color: Deep red (#C00000) to evoke the richness and freshness of the chicken.
- Background color: Light red (#FAD9D9), a gentle off-white to let the product photos stand out and convey freshness.
- Accent color: Darker red (#800000) to highlight key actions and navigation, creating emphasis without overwhelming.
- Font: 'PT Sans', a sans-serif, will be used throughout the site because it can handle headlines or body text.
- Use clear, modern icons for categories and actions (e.g., shopping cart, user profile). Make the style geometric.
- Design a clean, grid-based layout for product listings, ensuring easy navigation and visual appeal.
- Incorporate subtle transitions and animations to enhance user interaction (e.g., adding to cart, loading new products).
