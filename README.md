## Overview

MBC Chicken Express is a modern poultry e-commerce platform designed to streamline the ordering and delivery of fresh chicken products.

The platform enables customers to browse products, manage their cart, schedule deliveries, place orders securely, and track order status through an intuitive user experience optimized for both mobile and desktop devices.

---

## Features

### Product Catalog

* Category-based product browsing
* Multiple weight variants

  * 500g
  * 1kg
  * 1.5kg+
* Dynamic market-based pricing
* Product images and descriptions

### Shopping Cart

* Add and remove products
* Quantity management
* Real-time price calculation
* Order summary preview

### Checkout System

* Delivery address management
* Cash on Delivery (COD)
* UPI Payments
* Debit/Credit Card Payments
* Order confirmation workflow

### Scheduled Deliveries

Customers can choose delivery slots:

* Morning: 7:00 AM – 12:00 PM
* Evening: 6:00 PM – 9:00 PM

### Order Tracking

* View order history
* Track order status
* Delivery progress updates

### Customer Reviews

* Product ratings
* Customer feedback
* Review management

### Smart Recommendations

* Cart-based suggestions
* Purchase history recommendations
* Related product discovery

## Installation

```bash
git clone https://github.com/SanjibBayen/MBC_Chiken.git

cd MBC_Chiken

npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
